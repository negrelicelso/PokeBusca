<h1 align="center">PokeBusca</h1>

## Sobre o Projeto

O PokeBusca é um aplicativo feito com o intuíto de aprendizado particular. Com funcionalidades de consumo de API's, criação de GUI interativa pelo usuário, e programação utilizando padrão MVC.

## Sobre o Aplicativo

O PokeBusca é um sistema que utiliza uma API para fazer diversos tipos de buscas e exibição das informações na tela para fácil consulta.

## Tecnologias Utilizadas

- .NET 8.0
